# 🚗 SmartFuel – Fuel Consumption Prediction

## 📌 Description
SmartFuel predicts vehicle fuel consumption using Random Forest. It helps improve efficiency and reduce costs.

## 🎯 Objectives
- Accurate fuel prediction  
- Improve efficiency  
- Reduce costs  

## 🧠 Model
- Random Forest Regressor  

## ⚙️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Flask  

## 🚀 How to Run
git clone https://github.com/chakrika05/SmartFuel-.git  
cd SmartFuel-  
pip install -r requirements.txt  
python app.py
